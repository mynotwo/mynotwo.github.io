欢迎来到我的博客。这里会记录一些关于 **神经压缩 (neural compression)**、**系统效率** 和 **AI 基础设施** 的随手笔记——有时是论文读后感，有时是工程踩坑，有时只是一点想法。

## 这个博客怎么运作

- **中英双语**：每篇文章在 front matter 里标 `lang: zh` 或 `lang: en`，列表页可以按语言筛选，中文正文用了更舒服的行距和字体。
- **评论**：底部的评论区由 [Giscus](https://giscus.app) 驱动，数据存在本仓库的 GitHub Discussions 里——完全归我所有，你用 GitHub 账号就能留言、点赞。
- **RSS**：右上角有订阅入口，欢迎 follow。

## 写一篇新文章

往 `_posts/` 丢一个 `YYYY-MM-DD-slug.md` 就行：

```yaml
---
title: "标题"
date: 2026-06-08
lang: en
tags: [compression]
excerpt: "一句话摘要。"
---
正文……
```

代码高亮、公式、表格都照常工作。Happy writing!
